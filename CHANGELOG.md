# Changelog

## [0.2.0] - 2026-01-19

### Added
- Add age column with ring segment indicator to active spots
- Use filter-at-display-time and add 1-minute max age option
- Decouple VFD preview state from serial connection
- Filter spots at display time instead of insert time
- Add age_seconds and age_fraction methods to AggregatedSpot

### Changed
- Rename package from rbn-vfd-linux to rbn-vfd

## [0.1.0] - Initial Release

- Initial release with RBN telnet client, spot aggregation, and VFD display
